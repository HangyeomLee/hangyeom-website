# Hangyeom Portfolio Pro v2

A polished Next.js portfolio for job applications, with a recruiter-friendly structure and animation.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`

## Build for production

```bash
npm run build
npm run start
```

## Files you will edit most often

- `components/portfolioData.ts` → text, projects, links, skills, experience
- `components/PortfolioPage.tsx` → layout and sections
- `components/portfolio.module.css` → styling
- `app/layout.tsx` → metadata / browser title
- `public/Resume.pdf` → optional resume file if you want a direct download button
